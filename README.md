## Usage

```sh
vscpp help
```

```sh
vscpp examples
```

```sh
vscpp create:project WideHard
```

```sh
vscpp create:class foo
```

```sh
vscpp create:class src/foo/Bar
```

```sh
vscpp create:header foo
```

```sh
vscpp create:cpp bar
```
